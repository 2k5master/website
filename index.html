OOTP HTML 2 CSV CONVERTER

Using the Python BeautifulSoup4 library, I made a tool to convert Out of The Park Baseball's html files, and extract a table from it.


The tool will then analyze the stats and it will look for undervalued players. You can also input a college draft class and it will give you a recommened targets list.
